# Assignment 3: Module 4 - Functions & Modules in Python

## Description
This repository contains two Python scripts submitted for Assignment 3. Both scripts have been rigorously tested to handle user validation and math domain errors.

### Task 1: Calculate Factorial Using a Function
This script defines a custom function named `factorial` that calculates the factorial of a given number using recursion. It then returns the calculated factorial. Basic error handling is included to prevent infinite recursion if a negative number is entered.

### Task 2: Using the Math Module for Calculations
This script asks the user for a number as input and utilizes Python's built-in `math` module to calculate and display the following:
* The square root of the number.
* The natural logarithm (log base e) of the number.
* The sine of the number (calculated in radians).

---

## How to Run
1. Ensure you have Python 3 installed on your machine.
2. Extract the submitted ZIP folder and open it in your terminal or command prompt.
3. To run **Task 1**, execute: `python factorial.py` *(Note: replace with your actual file name)*
4. To run **Task 2**, execute: `python math.py` *(Note: replace with your actual file name)*
