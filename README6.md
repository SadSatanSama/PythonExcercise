# Assignment: CALCULATOR USING TKINTER

## Description
The folder contains a demo of a simple calculator using the python Library Tkinter.

## Workflow

### Task: Create a calculator using Tkinter Library (`data_structure.py`)
A simple calculator has been shaped using the Tkinter library. The library has been imported and using `title`, `geometry` and `label`, the rough structure has been generated. An output space has been created using the `entry` class. `Button` class has been used to create buttons for `numbers` and `arithmatic operations` and each operations (including `equal`) has been defined with global variables. The `mainloop`  has been added at the end to keep the calculator running. 

---

## How to Run
1. Ensure Python 3 is installed.
2. Open the terminal in the same folder where the code is.
3. For Task 1, run: `python tkinter_practice.py`
