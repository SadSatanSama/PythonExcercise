# Assignment: Data Structures & Strings in Python

## Description
The folder contains two different codes related to the usage of membership operator on dictionary and usage of slicing on a list.  

## Workflow

### Task 1: Create a Dictionary of Student Marks (`data_structure.py`)
A dictionary with some dummy data is created and user input is requested for any desired student's name and membership operator (`in`) is used to verify if the name is inside of the dictionary the corresponding marks is printed from dictionary when the name is is found as a key.

### Task 2: Demonstrate List Slicing (`list_slicing.py`)
range() function is used to create a list (my_list) with desired numbers. First 5 numbers are scooped out using slicing and assigned to a new list (extracted_list). Then the new list is reversed using slicing again (reversed_list) and printed.

---

## How to Run
1. Ensure Python 3 is installed.
2. Open the terminal in the same folder where the codes are.
3. For Task 1, run: `python data_structure.py`
4. For Task 2, run: `python list_slicing.py`
